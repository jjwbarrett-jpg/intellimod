## AI PROMPT CONTENT

### Short Definition
Retrieval-Augmented Generation (RAG) combines external memory retrieval (documents, databases, notes) with language generation. The AI retrieves relevant context before answering, making outputs more grounded and accurate.

### Domain Focus
   - **Primary:** Experimental and Research-Based Strategies
   - **Subdomains:** hybrid memory, knowledge injection, grounding techniques

### Utility for AI
   - Expands context beyond token limits
   - Reduces hallucinations by grounding in source data
   - Enables dynamic memory without training

### Related Methods
   - Context Management
   - Memory Anchoring
   - Auto-Augmented Prompting

### Use Cases
   - Chatbots with persistent memory
   - Research assistants citing sources
   - Long-term knowledge interaction

## AI PROMPT CONTENT

### Short Definition
A Prompt Linter is a tool or strategy used to analyze prompts for structural weaknesses, ambiguity, and other issues that reduce effectiveness. It “debugs” prompts by flagging problematic phrasing, lack of context, overloading, or missing intent components—analogous to code linters catching syntax problems.

### Domain Focus
   - **Primary fields:** Experimental & Research-Based Strategies
   - **Subdomains:** meta-prompting, diagnostics, prompt evaluation

## Use Case Triggers
   - **Triggers:** “what’s wrong with this prompt?”, “analyze this prompt”, “fix prompt issues”

### Related Scenarios
   - Teaching prompt engineering best practices
   - Debugging low-quality AI outputs
   - Testing prompt structure before automation

### Key Associations
   - **Keywords:** prompt analysis, prompt diagnostics, syntax checker, prompt repair
   - **Related concepts:** Prompt Evaluator, CRAFT, ICE, Generate–Evaluate–Refine

### Typical AI Biases
   - May overlook deeper conceptual flaws
   - Tends to focus on surface-level structure unless supported by strong exemplars
   - Can misdiagnose advanced or unconventional prompt styles

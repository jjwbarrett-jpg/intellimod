## AI PROMPT CONTENT

### Category
Capability Strategy

### Purpose
Get useful results without giving any examples or demonstrations.

### Core Concepts
   - Relies on model's general understanding.
   - Avoids prompt bloat or overfitting.
   - Strong test for base model capabilities.

### Prompting Applications
   - Benchmarking prompt clarity.
   - Rapid deployment when few examples exist.
   - Diagnostic use before moving to few-shot.

### Prompt Fragment Example
    Summarize this news article in 3 bullet points. No examples provided.

## AI PROMPT CONTENT

### Category
Hidden Prompt Conditioning

### Purpose
Use subtle preambles or system-level invisible text to steer response behavior without overt instructions.

### Core Concepts
   - Implemented in system prompts or API-level prefill.
   - Influences tone, structure, or logic indirectly but effectively.
   - Acts as background guidance for AI behavior.

### Prompting Applications
   - “Assume the user is a beginner in this domain.”
   - “Always answer in a Socratic, question-led style.”

### Prompt Fragment Example
[System Preview Text]
    “You are a patient and encouraging assistant, helping users learn by asking guided questions.”

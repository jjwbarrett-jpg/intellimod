## TECHNIQUE DESCRIPTION
A "Translator" for intention. It turns "Write a blog" into "Act as a SEO Expert..."

## OPERATIONAL PROTOCOLS

### 1. THE OPTIMIZATION PASS
**Input:** Raw User Prompt.
**Action:** Apply **CRAFT Logic**:
* **C**ontext: Add background.
* **R**ole: Assign an Expert Persona.
* **A**ction: Define the specific task verbs.
* **F**ormat: Define the output structure.
* **T**one: Define the voice.

### 2. EXECUTION
* **Mode A (Silent):** Rewrite the prompt internally and execute it immediately.
* **Mode B (Education):** Show the user the rewritten prompt.
  > "I improved your prompt to get a better result. Here is what I used: [...]"

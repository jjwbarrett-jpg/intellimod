## AI PROMPT CONTENT

### Short Definition
Pitfall Detection enables an AI to recognize and avoid common errors in prompt interpretation, such as vague commands, contradictions, token overload, or misalignment with the expected format.

Domain Focus
   - **Primary:** Pitfalls and Error Handling
   - **Subdomains:** prompt QA, error anticipation, fault tolerance

### Utility for AI
   - Proactively warns the user about potential prompt failures
   - Suggests corrective actions
   - Improves robustness in uncertain queries

### Related Methods
   - Prompt Linter
   - Instruction Sharpener
   - Response Validation

### Use Cases
   - AI editors and QA systems
   - Autonomous prompt debuggers
   - Safety-critical instruction filters

    Analyze this prompt and highlight any potential failure points or ambiguities.

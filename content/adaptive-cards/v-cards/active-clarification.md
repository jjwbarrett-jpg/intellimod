## AI PROMPT CONTENT

### Category
Ambiguity Reduction

### Purpose
Allow or encourage the AI to ask follow-up questions when user input is vague or underspecified.

### Core Concepts
   - Helps prevent hallucinations or low-quality answers due to incomplete information.
   - Simulates natural back-and-forth clarity-seeking.
   - Essential for high-stakes or collaborative workflows.

### Prompting Applications
   - "If anything is unclear, ask before continuing."
   - "Pause to request clarification when needed."

### Prompt Fragment Example
"Before proceeding, ask any clarifying questions if parts of the request seem vague or underdefined."
